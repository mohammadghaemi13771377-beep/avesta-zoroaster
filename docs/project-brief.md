# دستور فنی پروژه AVESTA-ZOROASTER

این پروژه نباید شبیه وبلاگ یا سایت مذهبی سنتی باشد. باید شبیه یک جهان دیجیتال سینمایی برای اوستا، زرتشت و خرد ایران باستان طراحی شود.

## Tech Stack

- Next.js 14+
- TypeScript
- Tailwind CSS
- App Router
- RTL-first Persian UI
- PostgreSQL
- Prisma ORM
- Meilisearch برای جستجوی سریع
- Strapi CMS یا پنل اختصاصی

## Brand

- نام: `AVESTA-ZOROASTER`
- دامنه: `avesta-zoroaster.com`
- شعار: `پندار نیک | گفتار نیک | کردار نیک`
- حس: باشکوه، رازآلود، خردمندانه، تاریخی، الهام‌بخش، مدرن، ایرانی، معنوی

## Design

- Dark Luxury Persian Mythology
- پس‌زمینه مشکی/سرمه‌ای
- accent طلایی
- کارت‌های سینمایی
- تصویرسازی AI اختصاصی برای هر بخش
- صفحه اول خلوت و دروازه ورود، نه فهرست سنگین همه متن‌ها
