# کتابخانه پرامپت‌های AI

هدف کتابخانه پرامپت این است که تصویرهای AVESTA-ZOROASTER یک‌دست، سینمایی و برندمحور تولید شوند.

## فایل اصلی

`lib/ai-prompts.ts`

این فایل برای هر فضای تصویری یک template نگه می‌دارد:

- عنوان
- بخش مرتبط
- mood
- رنگ accent
- مسیر خروجی پیشنهادی
- prompt
- negative prompt
- محل استفاده

## API

```http
GET /api/media/prompts
GET /api/media/prompts?section=yasna
```

## بخش‌های آماده

- Hero صفحه خانه
- یسنا
- گات‌ها
- وندیداد
- یشت‌ها
- خرده اوستا
- هات‌ها

## استفاده در پنل

در `/admin/media` بخش «کتابخانه پرامپت‌های تصویری» نمایش داده می‌شود. هر prompt قابل کپی است. فرم «ثبت رسانه» هم یک dropdown دارد که با انتخاب هر template، فیلدهای مربوط به تصویر پر می‌شوند.

## قانون سبک تصویری

برای حفظ هویت برند، در تصویرها این موارد باید تکرار شوند:

- پس‌زمینه تیره، مشکی عمیق یا سرمه‌ای سلطنتی
- نور طلایی کنترل‌شده
- معماری الهام‌گرفته از ایران باستان
- خط اوستایی یا مه نوشتاری محو
- حس موزه دیجیتال، نه تصویر فانتزی شلوغ

از این موارد پرهیز شود:

- لباس مدرن
- ساختمان مدرن
- موجودات فانتزی
- متن ناخوانا یا واترمارک
- شلوغی بیش از حد
